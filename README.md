# Robot Pet Project

A multi-page static documentation site for converting a Samsung SR8855 robot vacuum into an autonomous AI-powered robot pet.

## Project Structure

- `blog/index.html` - Overview and roadmap
- `blog/phases.html` - Build phases and timelines
- `blog/hardware.html` - Bill of materials
- `blog/power.html` - Power architecture
- `blog/wiring.html` - Wiring and GPIO reference
- `blog/software.html` - Software stack and code examples
- `blog/decisions.html` - Architecture decision records
- `blog/styles.css` - Shared visual design and responsive layout

## Quick Start

1. Open `blog/index.html` in a browser.
2. Navigate between pages using the sidebar nav (mobile uses a horizontal top nav).
