Expert review additions — Phase 1: The following components were identified as missing from the original Phase 1 BOM after independent engineering review. Several are critical safety items that must be in place before first power-on.
ComponentSpec / NotesSupplierPricePriority
Electrolytic capacitor 470µF–1000µF 35V+ VMOT decoupling capacitor. Placed directly across MDD10A VMOT and GND terminals. Absorbs motor back-EMF spikes that would otherwise destroy the driver. This is the single most important addition to the BOM. Observe polarity when fitting. AliExpress / RS Components ~£0.50 🔴 Critical
Emergency stop button (NC pushbutton) Normally-closed pushbutton wired in series with motor battery supply. Instantly cuts motor power when pressed. Must be mounted accessibly on chassis exterior. Essential for development and family safety. AliExpress / Amazon ~£2 🔴 Critical
INA219 current sensor modules (×2) I2C current sensing, one per motor supply line. Enables motor stall detection (current spike without velocity change), battery current monitoring, and runtime estimation. The MDD10A has no current sensing — these fill the gap. AliExpress ~£3 Significant
Pi 4 heatsink kit Heatsink set for Pi 4 SoC, RAM, and USB controller. Essential for Phase 2+ when running YOLOv8-nano continuously. Without cooling the Pi will thermal throttle, reducing vision frame rate significantly. Get the aluminium + copper kit, not just stickers. Pimoroni / Amazon ~£4 Significant
JST-PH connector kit (2mm pitch) For all signal connections (sensors → Pi). Keyed so can't be misinserted. Makes the build maintainable and sensor harnesses unpluggable during chassis work. Adopt from Phase 1 — retrofitting connectors later is painful. AliExpress ~£5 Significant
Voltage divider resistors (10kΩ + 20kΩ, pack) For 5V→3.3V signal level shifting on any OEM sensor found to output 5V. Required if sensor voltage verification (see Wiring page) reveals 5V outputs. Essential before connecting sensors to Pi GPIO — 5V signals destroy the GPIO bank. AliExpress (included in breadboard kit) ~£0 🔴 Critical
